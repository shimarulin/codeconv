# `@codeconv/codeconv`

> CodeConv CLI core

## Install

Using npm:

```bash
npm install @codeconv/codeconv
```

or using yarn:

```bash
yarn add @codeconv/codeconv
```

## Usage

CodeConv CLI can be extended with plugins. CodeConv plugin system will resolve plugins automatically, follow by this
naming convention:

- `@codeconv/plugin-*`
- `codeconv-plugin-*`
